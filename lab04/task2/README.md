### Project Overview
A refactored article management system with reusable components.

### Components
- **ArticleManager**: Parent container with articles state
- **AddArticle**: Form component for adding articles
- **ArticleList**: Maps through articles and renders ArticleItem
- **ArticleItem**: Individual article with local toggle state

### Features
- Add new articles with title and summary
- Remove articles
- Toggle article summary visibility
- Local state management in ArticleItem

### How to Run
1. npm install
2. npm start
