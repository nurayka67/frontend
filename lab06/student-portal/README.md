### Route Structure
- `/` - Home page with welcome message
- `/courses` - Courses listing page
- `/about` - About university page
- `/*` - 404 Not Found page

### Features
- Shared Layout with navigation and footer
- Client-side routing with React Router v6
- Link components instead of anchor tags
- TypeScript with strict mode
- Responsive CSS design

### Hooks Used
- createBrowserRouter for route configuration
- Outlet for nested routes
- Link for navigation
